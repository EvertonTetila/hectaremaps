# HectareMaps

Plataforma para processamento e análise de imagens obtidas por drones, com foco em aplicações de agricultura de precisão, geotecnologias e inteligência artificial.

O projeto integra técnicas de visão computacional, aprendizado de máquina e geoprocessamento para apoiar o monitoramento agrícola, a geração de mapas temáticos e a tomada de decisão no campo.
